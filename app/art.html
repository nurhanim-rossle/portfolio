Emblem Arcana

2018

A community project to create a colletion of 78 tarot cards, featuring around 80 artists working on the project. The profits made by the project was donated to CURE Childhood Cancer.

The project was a challenge to me as I need to decipher the meaning behind the Eight of Swords card. I made several drafts but finally settled with this pose as I think it fits the card's meaning best.

I really enjoyed working on this project and was grateful to be able to be part of it.

Al

2018

Al is the main protagonist from a series called Fire Emblem: Hasha no Tsurugi.

This is a side project where I created illustrations imitating the illustration in official Fire Emblem Heroes game. The illustration consists of four (4) different poses (normal, attack, special, damage).

FEH Chibi Project

2017-current

It is an ongoing personal project for me, in which I want to illustrate all character from the game Fire Emblem Heroes into their minified version.

I'm doing this project whenever I have my free time, which is why the progress are slower than my other projects.
